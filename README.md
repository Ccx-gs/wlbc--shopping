# wlbc--shopping

## Monorepo Structure

- `/frontend`: Vue 3 + Vite + Element Plus + Axios + Pinia
- `/backend`: Node.js + Express + CORS

## Run Backend

```bash
cd backend
npm install
npm run start
```

Backend API runs on `http://localhost:3000`.

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend app runs on `http://localhost:5173`.
